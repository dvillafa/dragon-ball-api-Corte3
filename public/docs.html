Dragon Ball API - Docs

Introducción

La Dragon Ball API es una API pública para acceder a datos de los personajes, técnicas, habilidades, y más del universo Dragon Ball.

Endpoints

GET /api/characters

Obtiene todos los personajes disponibles con opciones de filtrado y paginación.

{
  "characters": [
    {
      "id": "673bd22984a9b8c3714a3f53",
      "name": "Gohan (Adolescente)",
      "race": "Saiyan Hibrido",
      "techniques": ["Supermasenko", "Descarga de ráfaga de energía", "Superkamehameha"],
      "specialAbilities": ["Sentido salvaje", "Carga maxima"],
      "transformations": ["Super Saiyan", "Super Saiyan 2"]
    },
    {
      "id": "673bcb0584a9b8c3714a3f2c",
      "name": "Goku (Z-Temprano)",
      "race": "Saiyan",
      "techniques": ["Kamehameha", "Ataque de Kaioken", "Genkidama"],
      "specialAbilities": ["Taiyoken", "Kaioken"],
      "transformations": []
    }
  ]
}

GET /api/characters/:id

Obtiene los detalles de un personaje específico por ID.

{
  "id": "673bd22984a9b8c3714a3f53",
  "name": "Gohan (Adolescente)",
  "race": "Saiyan Hibrido",
  "techniques": ["Supermasenko", "Descarga de ráfaga de energía", "Superkamehameha"],
  "specialAbilities": ["Sentido salvaje", "Carga maxima"],
  "transformations": ["Super Saiyan", "Super Saiyan 2"]
}

GET /api/characters?name={nombre}

Obtiene personajes filtrados por el nombre. Ejemplo: puedes buscar por "Goku" o "Gohan".

{
  "characters": [
    {
      "id": "673bd22984a9b8c3714a3f53",
      "name": "Gohan (Adolescente)",
      "race": "Saiyan Hibrido",
      "techniques": ["Supermasenko", "Descarga de ráfaga de energía", "Superkamehameha"],
      "specialAbilities": ["Sentido salvaje", "Carga maxima"],
      "transformations": ["Super Saiyan", "Super Saiyan 2"]
    }
  ]
}

GET /api/characters?race={raza}

Filtra personajes por su raza. Ejemplo: "Saiyan", "Humano", etc.

{
  "characters": [
    {
      "id": "673bd6f184a9b8c3714a3f67",
      "name": "Krillin",
      "race": "Humano",
      "techniques": ["Kamehameha", "Onda de energia expansiva", "Kienzan en cadena"],
      "specialAbilities": ["Taiyoken", "Ataque de Ilusion de imagen"],
      "transformations": []
    }
  ]
}

GET /api/characters?technique={tecnica}

Filtra personajes por técnica. Ejemplo: puedes buscar por "Kamehameha", "Genkidama", o "Supermasenko". 

{
  "characters": [
    {
      "id": "673bd22984a9b8c3714a3f53",
      "name": "Gohan (Adolescente)",
      "race": "Saiyan Hibrido",
      "techniques": ["Supermasenko", "Descarga de ráfaga de energía", "Superkamehameha"],
      "specialAbilities": ["Sentido salvaje", "Carga maxima"],
      "transformations": ["Super Saiyan", "Super Saiyan 2"]
    }
  ]
}
